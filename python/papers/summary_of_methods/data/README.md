## Files

### `calibration_data.json`
**What it is:** Full calibration sweep results.
**How generated:** `scripts/calibration_sweep.py`
**Contents:** List of records, one per (N, q) pair.
Each record contains:
- `N` — perimeter node count
- `q` — area stiffness ratio
- `q_b` — product q·b
- `eps_arr` — array of per-particle strains recorded during squeeze
- `dA_arr` — fractional area loss (%) at each frame
- `nu_arr` — measured Poisson ratio at each frame
- `metrics` — dict keyed by ε_ref value (`"0.04"` through `"0.12"`),
  each containing `{"nu": float, "dA": float}`
- `cc_pct` — maximum contact compression (%) as a quality indicator

**Feeds:**
- Table I (18-point calibration, N=48, ε_ref=0.08)
- Table II (N-sensitivity)
- Fig. 2 (ΔA-ν trajectory)
- Fig. 3 top (N-sensitivity curves)
- Fig. 3 bottom (ε_ref-sensitivity curves)
- Fig. 4 (N-drift)

**Sweep parameters:**
- N ∈ {32, 48, 72}
- q ∈ {0.05, 0.10, 0.15, 0.20, 0.30, 0.50, 0.75, 1.00, 1.50, 2.00,
       3.00, 5.00, 7.50, 10.0, 15.0, 20.0, 30.0, 50.0}  (18 values)
- ε_ref ∈ {0.04, 0.06, 0.08, 0.10, 0.12}
- Fixed: b=0.2, τ=√2.4≈1.549, S=1, R0=1, α₀=2.0, SR=0.01, C₀=3000

### `calibration_table.csv`
**What it is:** Flat CSV version of `calibration_data.json` for easy inspection.
**How generated:** `scripts/calibration_sweep.py` (same run)
**Columns:** `N, q, q_b, eps_ref, nu, dA, cc_pct`
One row per (N, q, ε_ref) combination — 3 × 18 × 5 = 270 rows.

---

### `fixed_b_results.json`
**What it is:** Early fixed-b sweep at N=32 over a coarser q grid.
**How generated:** `scripts/fixed_b_q_sweep.py`
**Contents:** List of records per q value containing squeeze trajectory
arrays and summary metrics.
**Note:** This was the initial exploration sweep (N=32 only, 10 q values).
The definitive calibration is `calibration_data.json`.

**Feeds:**
- Fig. 1 top (squishiness axis overview)
- Fig. 1 bottom (perimeter shape overlays)
- Supplemental Movies S2–S6 (source GIFs generated by same script)

---

### `R0_spot_check.json`
**What it is:** R0 invariance verification results.
**How generated:** `scripts/R0_spot_check.py`
**Contents:** List of records, one per (ν_target, R0) combination.
Each record contains:
- `nu_target`, `q`, `R0`, `EI`
- `nu_meas`, `dA_pct`, `cc_pct`
- `params` — full parameter dict used

**Feeds:**
- Table III (R0 spot-check)
- Fig. 5 (R0 invariance plot)

**Sweep parameters:**
- ν_target ∈ {0.30, 0.60, 0.85}
- R0 ∈ {0.5, 1.0, 2.0}
- Calibration source: `calibration_data.json` at N=72, ε_ref=0.08
